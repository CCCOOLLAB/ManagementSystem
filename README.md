# 后台管理系统-Vue+Element Ui

## Project setup
```
版块功能：
1.登录版块
    1.1登录数据验证
    1.2登录数据重置
    1.3登录访问权限
2.菜单版块
    2.1获取菜单数据
    2.2菜单效果设置
3.用户管理-用户列表
    3.1获取用户数据
    3.2列表分页效果
    3.3修改用户状态
    3.4数据搜索功能
```

yarn install
